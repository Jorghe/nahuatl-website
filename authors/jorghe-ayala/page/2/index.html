    2. Reverencia y apreciación

    Para denotar el respeto hacia una persona u objeto, se puede agregar el prefijo Pil o el sufijo tzin (y se pueden combinar). La palabra Pilitzcuintli (Pil + Itzcuintli) se refiere a un perro muy venerado o apreciado. Existe otra forma aun más reverencial que sería xochitzitzintin (xochitl + tzitzintin) que denota un grado mayor de apreciación, y éste se puede combinar así también con Pilitzuintzitzintin (Pil + itzcuintli + tzitzintin) para hacer una forma más exagerada de apreciación.
